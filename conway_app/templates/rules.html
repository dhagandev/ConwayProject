Learn the Game of Survival

The Rules

  1. Each alive cell with one or no neighbors dies, as if by solitude.
  2. Each alive cell with four or more neighbors dies, as if by overpopulation.
  3. Each alive cell with two or three neighbors survives together.
  4. Each empty cell with exactly three neighbors becomes alive.
